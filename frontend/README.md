# Salon Queue App - React Frontend

A React-based frontend for salon/barbershop queue and appointment management.

## Tech Stack

- **React 18** + **Vite**
- **React Router v6**
- Plain CSS (no UI library)

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Features

### Customer
- Login (OTP, Password, Google, Guest)
- QR Scan / Search to find salon
- Salon Dashboard: Profile, Live Queue, Book Slot, Price List, Rating, Feedback
- My Appointments (Upcoming + Past)

### Vendor
- Home with chart placeholders
- Booking & Queue (Live queue, Slot availability, Walk-in)
- Payment & Billing
- Communication (Chat, Appointments, Broadcast)
- Reports (Ratings, Customers, Earnings)
- Gallery & Branding
- Settings & Support
- My Profile (Business, Services, Staff, Discounts)
